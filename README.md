# mathieux51/homebrew-tap

Homebrew tap for mathieux51's projects.

## Installation

```bash
brew tap mathieux51/tap
```

## Available Casks

### Vimtico

Native macOS PostgreSQL client with Vim mode and Nord theme.

```bash
brew install --cask mathieux51/tap/vimtico
```

See [Vimtico repository](https://github.com/mathieux51/vimtico) for more details.
